# Dream Assets

This directory contains assets for the horse dream system.

## Structure

### `/settings/`
Add your dream setting background images here (e.g., beach scenes, forest landscapes, meadows, etc.)

Supported formats: `.png`, `.jpg`, `.jpeg`

Example files you can add:
- `beach.png` - A peaceful beach scene
- `forest.jpg` - A magical forest setting
- `meadow.png` - A sunny meadow landscape
- `mountains.jpg` - Majestic mountain vista
- `sunset.png` - Beautiful sunset scenery

### Usage

The dream system will automatically detect and use any images you place in the `/settings/` folder. Simply:

1. Add your dream background images to `/settings/`
2. The dream composer will randomly select from available images
3. If no custom images are available, the system falls back to CSS gradients

### Guidelines

- Use high-resolution images (at least 1280x720) for best quality
- Choose calming, dreamy scenes that fit the peaceful nature of horse dreams
- Images should work well with the mood filters (happy, peaceful, magical, etc.)
